# Print Rsassy feature information

Print Rsassy feature information

## Usage

``` r
# S3 method for class 'sassy_features'
print(x, ...)
```

## Arguments

- x:

  A `sassy_features` object returned by
  [`sassy_features()`](https://sounkou-bioinfo.github.io/Rsassy/reference/sassy_features.md).

- ...:

  Ignored; accepted for compatibility with
  [`print()`](https://rdrr.io/r/base/print.html).

## Value

`x`, invisibly.
